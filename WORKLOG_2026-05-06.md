# 2026-05-06 开发日志

## 1.11.6 · 移动端商品与仓库卡片改造

- 商品管理移动端列表改为与销售记录一致的卡片式信息流，标题展示型号，副标题按“售价 / 数量 / 属性”和“品牌 / 系列 / 总销量”两行组织。
- 商品卡片点击后打开同款上滑二级菜单，提供商品详情、商品分布、编辑和删除入口；编辑/删除仅对具备管理员或商品权限的账号显示。
- 商品详情移动端改为订单详情同款信息网格，集中展示 ID、型号、品牌、系列、属性、售价、数量、总销量、条码和更新时间。
- 店铺/仓库管理移动端把店铺、仓库、其他仓库、维修点四类列表改为同款卡片；标题为地点名称，副标题展示商品数量、店长/库管/工程师、地址与联系方式。
- 店铺/仓库卡片点击后打开同款上滑操作菜单，并移植桌面端“操作”入口：商品、排班、目标、编辑、删除按原有权限和地点类型显示。
- 新增移动端专用样式只在小屏隐藏主表格，桌面表格和弹层内部表格不受影响；所有新增卡片继续使用现有主题变量，适配深色模式。
