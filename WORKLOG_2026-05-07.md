## 1.11.8 · 安全匹配商品图片与移动端销售确认优化

- 重新检查本次问题原因：上一轮图片处理混入了商品导入语义，来源范围过宽且曾产生重复型号；本次改为只读旧系统 `clock_offline`，只按型号匹配图片，不再创建或改动商品基础数据。

- 本地生成的导入 SQL 仅包含 977 条 `UPDATE aqc_goods_items SET cover_image = ..., image_list = ... WHERE id = ...`，不包含 `INSERT`、`DELETE`，不触碰库存、销售、库存日志和工单表。
