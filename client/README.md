# AQC-N Client

## 本地运行

```bash
cd client
npm install
npm run dev
```

默认端口：`5173`

可选环境变量：

```bash
VITE_API_BASE=http://127.0.0.1:8010/api
```

## 页面说明

- 顶栏：左侧品牌 `AQC-N`，中间功能入口，右侧用户名悬停菜单（用户中心/退出登录）
- 欢迎页：TradingView 风格销售趋势图，支持 `1天 / 1周 / 1月 / 今年开始到现在` 切换
- 模块页：按 AQC-O 历史分组先做入口铺设，后续逐项迁移
- 用户中心：查看身份信息与登录会话，支持刷新登录态/下线会话
